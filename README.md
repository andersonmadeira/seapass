## 🌊🔐 Seapass

Password management _powered by ⚛️ React_

### Instructions

#### Install dependencies

`yarn`

#### run on dev mode

`yarn start`

#### build for prod

`yarn build`

#### tests (todo)

`yarn test`

#### lint code

`yarn lint`

#### pretty code

`yarn pretty`
